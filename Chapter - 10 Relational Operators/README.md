# 1 - Less Than
```
public class LessThan {
    public static void main(String[] args) {
        int x = 1;

        int y = 10;

        System.out.print("x = ");

        System.out.println(x);

        System.out.print("y = ");

        System.out.println(y);

        System.out.println(x + " < " + y + " = " + (x<y));
    }
}
```

Output
```
x = 1

y = 10

1 < 10 = true
```
